# 模块: 标签空文件检查

目标: 验证 labels 文件是否为空, 导致训练读取不到实例。

## 最小任务清单
- [x] 抽样打开 labels/*.txt, 检查是否为空或只有空行。
- [x] 统计空文件数量与占比。
- [x] 记录空文件样本清单与比例。
- [x] 若空文件占比为 0, 标记通过。

## 验证记录
- train: 12 个空文件 (示例: 0408.txt, 0409.txt, 0562.txt, 0563.txt, 0615.txt)
- val: 2 个空文件 (示例: 0258.txt, 0469.txt)
- test: 3 个空文件 (示例: 0169.txt, 0231.txt, 0276.txt)
- 结论: 空标签存在, 可能导致部分样本被视为背景

## 处理建议(不改数据)
- 重新导出标注, 确保所有目标写入标签文件
- 或将无目标样本显式标记为背景并在训练中允许背景样本
