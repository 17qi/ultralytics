# 项目需求文档（Project Brief）

## 项目名称
YOLO11 + RGB-D 生菜/杂菜 ROI 茎点定位

## 背景与目标
本项目使用当前工程目录中的数据与脚本，完成基于 RGB-D 的生菜/杂菜 ROI 茎点定位。目标是产出可训练、可评估的模型方案，并用现有评估脚本进行验证。

## 已有输入
- RGB 图像数据
- Depth 图像数据
- YOLO 标签
- 模型 YAML
- 评估脚本

## 任务范围
- 训练基于 YOLO11 的茎点定位模型
- 使用 RGB 与 Depth 信息进行建模或融合
- 通过现有评估脚本输出评估结果

## 预期输出
- 可复现的训练与评估流程说明
- 评估结果与关键指标汇总（由评估脚本生成）

## 约束与合规
- 只允许创建或修改 doc 目录下的 Markdown 文件
- 不修改任何 .py、.yaml、.txt、数据文件
- 不覆盖原始数据
- 不使用测试集选择阈值、模型或超参数

## 依赖与环境
- 依赖当前工程目录中现有脚本与配置文件
- 训练/评估的具体运行方式以仓库现有脚本为准

## 里程碑（最小可行）
1. 明确数据结构、标签含义与茎点定义
2. 明确训练与评估入口脚本及参数
3. 明确评估指标与验收标准
